# Painted scene layers

Drop generated art here as `<dir>/<layer>.webp` (and `<dir>/sky.webm` +
`sky.mp4` for the video beats). Folders: `hero/ roots/ rain/ dawn/ noon/
golden/ night/ cta/`. Every file is optional — the page upgrades as files
land, no code changes.

Exact filenames, prompts, sizes and budgets: `docs/landing-art-pipeline.md`.
Layer script: `src/components/landing/chapters/layerManifest.ts`.

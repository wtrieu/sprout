# Legacy painted backdrops (fallback only)

These seven Niji 7 skies now serve as FALLBACKS: the page looks for
`../scenes/<dir>/sky.webp` first and only uses these when the scenes/ file
is missing. New art goes in `../scenes/` — see `docs/landing-art-pipeline.md`.
Once every beat's scenes/ sky has landed, this folder can be deleted.

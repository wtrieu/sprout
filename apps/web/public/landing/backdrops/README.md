# Painted backdrops (optional)

Drop Midjourney-generated WebP backdrops here and the landing page uses them
automatically — any missing file keeps that chapter's procedural sky.

Expected filenames (see `docs/midjourney-prompts.md` for the full prompt pack
and workflow):

- `hero.webp` — beat 0, twilight underground
- `roots.webp` — beat 1, plum earth
- `rain.webp` — beat 2, storm sky
- `dawn.webp` — beat 3, sunrise
- `noon.webp` — beat 4, blue-sky meadow (**generate this one first** — it's
  the `--sref` style anchor for the rest)
- `golden.webp` — beat 5, golden hour
- `night.webp` — beats 6 + 7, moonlit night

21:9 aspect, WebP quality ~80, target ≤ 400 KB each.
